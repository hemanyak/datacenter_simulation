# datacenter_simulation
This is my attempt at understanding the issues behind water usage and energy consumption in AI data centers. I have created a simulation using Python which allows users to simulate exactly what happens in AI data centers, from workload to heat optimization to water usage. Feel free to try it out!

More specifics here ->
The project creates a basic simulation which demonstrates how an AI data center operates its energy usage and heat generation and water consumption during a full 24-hour duration. The system displays hourly and total energy consumption and thermal emission and water usage both at the facility and from external sources to demonstrate how large-scale AI operations impact the environment. The program produces different results with each execution because it uses random values for server usage and power usage effectiveness (PUE) and water usage effectiveness (WUE), which creates authentic variable data results. The simulation requires you to execute the Python file and use the dashboard which displays interactive charts that allow you to scroll and track data points and use the "Re-simulate" button to create fresh data. Users can identify system patterns through output analysis which demonstrates how increased energy consumption leads to greater heat and water requirements and reveals the amount of water needed to generate electricity and shows the system's effectiveness at managing water and heat resources for each AI token processed.
